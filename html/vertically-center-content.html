水平垂直居中

方法一：利用 translate

设置 父元素 position为 非 static， 容器元素 

                position: absolute;
                top: 50%;
                left: 50%;
                transform: translate(-50%, -50%);

我是需要居中的容器

方法二：利用 table

将 祖父元素设置成 display:table; width:100%， 父元素设置成 

                display: table-cell;
                vertical-align: middle;
                text-align: center;

我是需要居中的容器

方法三：利用 flex

将 父元素设置成 

                display: flex;
                justify-content: center;
                align-items: center;

我是需要居中的容器

方法四：利用 position

设置 父元素的 position为非 static，设置容器为 

                position: absolute;
                top: 0;
                bottom: 0;
                left: 0;
                right: 0;
                margin: auto;

此方法要求居中容器有 固定的宽高

我是需要居中的容器
